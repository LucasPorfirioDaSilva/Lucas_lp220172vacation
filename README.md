# Lucas_lp220172vacation

git clone - Clona um repositório em um novo diretório local
git init - Cria um repositório git vazio  ou reinicia um existente

git add - adiciona arquivo a lista de atualização do repositório
git log - mostra os logs dos commits /logs - registro de alteração
git status -  mostra o status da arvore atual 

git branch - lista , cria ou deleta galhos do repositório
git checkout - pula entre os galhos ou restaura galhos do repositório
git diff - mostra as diferenças entre commits e/ou repositório 

git pull - encontra e integra ao repositório local um repositório remoto 
git push - atualiza as referencias contidas no repositório
